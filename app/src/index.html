how it works
01 —

POST your URL

Send a POST request to /shorten with your long URL in the request body.

02 —

Hash & store

SHA-256 hashes your URL into a unique 8-character ID, stored instantly in DynamoDB.

03 —

Instant redirect

Visit your short link — the API looks it up and issues a 307 redirect in milliseconds.

api reference
bash — shorten a url 
# Shorten a URL
curl -X POST https://ahmedumami.click/shorten \
  -H "Content-Type: application/json" \
  -d '{"url": "https://your-long-url.com/example"}'

# Response
{
  "short": "a3f9b1c2",
  "url": "https://your-long-url.com/example"
}

# Use your short link — redirects automatically
curl -L https://ahmedumami.click/a3f9b1c2
